### :running: How to run

```bash

# Clone this repository
$ git clone https://github.com/femalves/ClientManager

# From the terminal, go to folder
$ cd clientmanager

# Install dependencies
$ npm install or yarn add

# Run Application
$ npm start ou yarn start

# Go to http://localhost:3000

```
